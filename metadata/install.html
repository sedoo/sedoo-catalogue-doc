npm run install:roll

# Principes

Le créations des modules EcmaScript (ESM) se fait en utilisant le bundler Rollup. Il permet une conversion des Single File Component (SFC) en modules EcmaScript (ESM) avec une grande simplicité sans modifier ses habitudes de codage hormis les quelques points précisés dans la section limitation. Pour aller plus loin dans RollUp : Doc officielle Vous n'avez pas à importer dans un SFC des modules ou librairies comme axios ou bien vuetify. Ils sont importés globalement dans le catalogue et seront récupérés par le bloc lors de leur instanciation dans le contexte du composant d'arrivé.
